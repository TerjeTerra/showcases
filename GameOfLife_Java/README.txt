-- GAME OF LIFE --
Et prosjekt skrevet som obligatorisk innlevering våren 2023.

Startes fra filen Hovedprogram (når kompilert).
Kan evt. oppgi egendefinert størrelse på vindu som parameter,
ellers starter standard oppsett 8 x 12 rutenett.

Spillet er en visualisering av Game of Life, der celler (@) vil 
formere seg eller dø etter bestemte regler basert på sine nabo-
celler. Man kan klikke på celler for å endre status (død eller
levende) når som helst. Spillet kan startes, pauses og avsluttes.
